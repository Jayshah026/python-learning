## if Statement 

- If we have a condition to follow we use if statement to show that.
- It works when we only have 1 condition to follow if it is true. 
----------------------------------------------------------------------------------------------------------------------------------------------------------------------
## if else ladder 

- When we have both True and False condition we use if else ladder.
- Here if contains the Truth statement, Means if the condition become true the if statement executes.
- else contains the flase statement, Means if the condition is not true then else statement will executes.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------
## if elif else ladder 

- When we have multiple condition to follow we use if elif else ladder.
- Here if will contain one condition.
- There can be multiple elif conditions.
- And at last if any of the condition will not execute then else block of code will executes.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------
