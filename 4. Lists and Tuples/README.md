## List 

- A list is a Mutable data type in which we can store multiple values.
- In list All values can have different data type.
- Just like string and arrays the list indexing start with 0th index.

- The main functions of Lists are :
1. .append() : Adds an item at the end of the list.
2. .insert(index, value) : Adds an item at a specific position.
3. .extend() : Adds multiple items from another list.
4. .remove(value) : Removes the first occurrence of a value.
5. .pop() : Removes and returns an element.
6. .index(value) : Returns the index of an element.
7. .count(value) : Counts how many times a value appears.
8. .sort() : Sorts the original list. 

- Etc.. all of them and more operations are included in the list.

## Tupple  

- A Tupple is an immutable data type and also here we can store multiple values just can not change it. 
- In tupples too the data type for the values can be different. 
- Here also the indexing is started form the 0th index. 

- The main functions of the Tupples are : 
1. .count(value) : Counts how many times a value appears in the tuple.
2. .index(value) : Returns the index of the first occurrence of a value.
3. len(tuple) : Returns the number of elements in the tuple.
4. in : Checks whether a value exists in the tuple.
5. + : Joins two tuples together.
6. * : Repeats a tuple multiple times.
7. tuple(iterable) : Converts another iterable (list, string, etc.) into a tuple.
8. min(tuple) : Returns the smallest element.
9. max(tuple) : Returns the largest element.
10. sum(tuple) : Returns the sum of all numeric elements.
11. sorted(tuple) : Returns a sorted list from the tuple.

- Etc.. all of them and more operations are included in the tupple.
