## Variables 

- Variables are the containers which hold values.
- Ex : a = 45, Here a is a variable and 45 is the value inside it.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
## Data Types

- In python we have main 5 data types
1. Integer : Store integer values. Ex : 4, 323, 43423, 63822232 etc..
2. Float : Store Decimal values. Ex : 4.33, 4232.34 etc..
3. String : Store Stirng in it. Ex : "Jay", "Heet", "Dhruv", "Hetva", "Mohit" etc..
4. Boolean : Store True or False.
5. None : To show that there is no value inside the variable we use None.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
## Operators 

1. Arithmetic Operators: +, -, *, /, % etc..
2. Assignment Operators: =, +=, -= etc..
3. Comparison OR Conditional Operators: == , >, >=, <, <=, != etc..
4. Logical Operators: and, or, not.
