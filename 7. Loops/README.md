## Loops 

- When a process is a repeating work then we use Loops. 
- It makes repeating work fast, reliable and accurate. 
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
##  while loop 

```Python

- Basic structure of while loop

i = 0
while(i < 11)
    print(i)
    i += 1

- Here i = 0 is a initial value in the while loop which will be outside of the loop.
- Then we have condition in while(), If the condition is true then only the loop will executes otherwise it will not execute.
- After that we can print whatever we want as per the requirement. 
- After that we apply the increment/decrement as per the condition. 
- This is how a "while loop" works.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
```
## for loop 

```Python
- Basic structure of for loop 

for i in range(1, 11):
    print(i)

- Here i is a variable in which we will store and print the value. 
- Now here in range(1, 11) -: 1 is a starting point here so i will start from the 1, and 11 is the ending point so 11 will not    print here it will simply exclude. 
- So here the loop will run for 1 to 10, 1 = starting point & 11 = Ending point which is not included. 
- Then we can print values easily with the use of print. 
- That's how for loop works
```
