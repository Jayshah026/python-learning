## Functions

- A function is a block of code which will perform a specific task. 
- It can be reused again and again. 

Syntax (Function definition) : 

def function_name():
    // Code

- Now whenever we want to use this block of code called function we will call it and it will execute and we will get an answer.

Calling a function (Function call):

def function_name():
    // Code

function_name() // This will call the function and now we can do the task we want. 

- This is how a function works.
