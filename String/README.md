## String Declaration 

- Here we can declare string in 3 ways..
1. name = "Jay"
2. name = 'Jay'
3. name = ''' Jay '''

- This are the ways by which we can declare a string.

## String Functions 

- Here to change in the string we use functions.
1. replace() - It replaces the specific word or words with the words that we want.
2. len() → Gives the length of the string.
3. upper() → Converts all characters to uppercase.
4. lower() → Converts all characters to lowercase.
5. capitalize() → Makes the first letter uppercase and the rest lowercase.
6. title() → Makes the first letter of every word uppercase.
7. find() → Finds the position (index) of a character or word. Returns -1 if not found.

- This are the some of the useful python string functions to use.

## Escape Characters

1. \n → New Line
2. \t → Tab Space
3. \\ → Print a Backslash
4. \" Variable \" → Double Quote Inside String
5. \' → Single Quote Inside String

- This are some of the useful Escape Characters we use in every language.
